This is a working website for a virtual gallery project. We are currently testing the spatial model, scene navigation, artwork placement, lighting and edit mode.

Galeria 3D

3D gallery

The top section of the website runs an interactive Babylon.js scene with the current gallery model.

Tryb edycji

Publiczny użytkownik ogląda galerię, a edytor po zalogowaniu może zmieniać układ prac.

Edit mode

Public visitors can view the gallery, while the logged-in editor can adjust the artwork layout.

Jeden aktualny stan

Po zapisaniu galeria pokazuje ostatni zatwierdzony układ — bez publicznej historii poprzednich wersji.

One current state

After saving, the gallery displays the latest approved layout — without a public history of previous versions.

Roadmapa projektu

Główne założenia rozwoju galerii.

Poniższa roadmapa pokazuje publiczny kierunek projektu. Szczegółowe poprawki techniczne pozostają częścią wewnętrznych etapów pracy.

Project roadmap

Main development goals for the gallery.

This roadmap presents the public direction of the project. Detailed technical improvements remain part of the internal production process.

Zrobione

  • Model galerii 3D
  • Symulacja chodzenia po scenie
  • Point & click oraz zbliżenie do prac
  • Umieszczanie obrazów na ścianach
  • Oświetlenie generowane przy obrazie
  • Kolorowanie ścian
  • Ekran ładowania

W testach

  • Stabilność kolizji
  • Clipping i granice ścian
  • Lepsze tekstury przestrzeni
  • Proporcje obrazów
  • Usprawnienie rotacji obiektów

Kolejne kroki

  • Rzeźby ustawiane na podłodze
  • Podstawki i standy ekspozycyjne
  • Wiszące oraz nietypowe formaty obrazów
  • Okrągłe obrazy i małe plansze
  • Lepsze narzędzia edytora

Pomysły

  • Wędrujący NPC
  • Interaktywny kustosz
  • Koncepcja schodów
  • Materiały filmowe w galerii
  • Rozbudowa narracji wystawy

Done

  • 3D gallery model
  • Scene walking simulation
  • Point & click with artwork zoom
  • Artwork placement on walls
  • Lighting generated for artworks
  • Wall color editing
  • Loading screen

In testing

  • Collision stability
  • Clipping and wall boundaries
  • Improved space textures
  • Artwork proportions
  • Object rotation improvements

Next steps

  • Sculptures placed on the floor
  • Display bases and stands
  • Hanging and non-standard artwork formats
  • Round artworks and small boards
  • Improved editor tools

Ideas

  • Walking NPC
  • Interactive curator
  • Staircase concept
  • Video materials inside the gallery
  • Expanded exhibition narrative
