Working version

Online 3D gallery prototype.

A working version of a virtual gallery, used to test spatial navigation, artwork presentation, lighting and exhibition editing tools.

Galeria 3D

Interaktywna przestrzeń pozwala oglądać układ galerii i prezentowane prace w środowisku 3D.

3D gallery

An interactive space for viewing the gallery layout and presented works in a 3D environment.

Tryb edycji

Odwiedzający widzą gotową galerię, a zalogowany edytor może układać prace i przygotowywać ekspozycję.

Edit mode

Visitors see the finished gallery, while the logged-in editor can arrange works and prepare the exhibition.

Jeden aktualny stan

Każdy zapis nadpisuje poprzedni układ. Odwiedzający zawsze widzą aktualną wersję galerii.

One current state

Each save replaces the previous layout. Visitors always see the current gallery version.

Budynek galerii

Cyfrowa wersja realnej przestrzeni.

Projekt opiera się na fizycznym miejscu galerii i rozwija je jako przestrzeń online do oglądania wystaw, testowania układu prac oraz tworzenia nowych form prezentacji.

Gallery building

A digital version of a real space.

The project is based on the physical gallery location and develops it as an online space for viewing exhibitions, testing layouts and building new presentation formats.

Roadmapa projektu

Rozwój galerii.

Główne obszary, nad którymi pracujemy w kolejnych etapach projektu.

Testowane

  • Wersja mobilna galerii
  • Interfejs edytora ekspozycji
  • Wgrywanie własnych prac przez edytora
  • Informacje o pracy i autorze w popupie
  • Kolizja ścian i poprawki clippingu

Kolejne kroki

  • Przebudowa systemu oświetlenia sceny
  • Segmentacja ścian pod optymalizację świateł
  • Inteligentne przypisywanie świateł do fragmentów ścian
  • Kolizja z obiektami, rzeźbami i obrazami
  • Wymiana tekstur i modeli galerii

Pomysły

  • Wędrujący NPC
  • Interaktywny kurator
  • Koncepcja schodów
  • Materiały filmowe w galerii
  • Rozbudowa narracji wystawy

Project roadmap

Gallery development.

The main areas we are working on in the next stages of the project.

In testing

  • Mobile gallery version
  • Scene lighting system
  • Lighting editor: Spot / Point
  • Exhibition editor interface
  • Online saving and loading of the current gallery state

Next steps

  • Uploading custom artworks by the editor
  • Dynamic artwork aspect ratio
  • Replacing gallery textures and models
  • Collision and clipping fixes
  • Displaying author and artwork information below artworks

Ideas

  • Walking NPC
  • Interactive curator
  • Staircase concept
  • Video materials inside the gallery
  • Expanded exhibition narrative
